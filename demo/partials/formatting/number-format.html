TW Number Format

twNumberFormat is a filter for formatting numbers based on a given locale. You can also supply a precision if you want a precise number of decimals. If precision is not supplied it will simply display any decimals the number value already has.

Internally this uses Number.prototype.getLocaleString, which is not yet supported in Safari.

Formatted number
{{ $ctrl.number | twNumberFormat : $ctrl.locale : $ctrl.precision }}
{{{{ $ctrl.number }} | twNumberFormat : "{{ $ctrl.locale }}" : {{ $ctrl.precision }} }}
