TW Fieldset

tw-fieldset is a component that allows us to build a simple form dynamically. The values of the form controls are mapped to the supplied model object. tw-fieldset is a companion to tw-definition-list which can be used to display the generated model.

Configuration
<tw-fieldset
  legend="Fieldset"
  fields="$ctrl.fields"
  model="$ctrl.model">
</tw-fieldset>
Model
{{$ctrl.model | json}}
