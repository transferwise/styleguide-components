TW Date Format

twDate and twDateTime are two Angular.js custom filter to format dates. The first one only shows the date while the second one displays both the date and the time. Dates older than 7 days are displayed without the weekday and dates from the same year are displayed without the year.

Date format filter

twDate is an Angular.js custom filter to format dates. It is useful for internationalization since different countries follow during date formats (e.g. dd/MM/yyyy vs MM/dd/yyyy vs yyyy/MM/dd) and time formats (12h vs 24h). The twDate filter takes 1 parameter: locale. The date can be either a Date object or a string.

Date filter (now)
{{$ctrl.dates.now | twDate:$ctrl.date.locale }}
Date filter (7 days ago)
{{$ctrl.dates.oneWeekAgo | twDate:$ctrl.date.locale }}
Date filter (2016)
{{'2016-02-03 15:20' | twDate:$ctrl.date.locale }}
<dl>
  <dt>Date filter</dt>
  <dd>{{ $ctrl.model | twDate:"{{$ctrl.date.locale}}" }}</dd>
</dl>
Date-time format filter

It is useful for internationalization since different countries follow during date formats (e.g. dd/MM/yyyy vs MM/dd/yyyy vs yyyy/MM/dd) and time formats (12h vs 24h). The twDateTime filter takes 1 parameter: locale. The date can be either a Date object or a string.

{{$ctrl.dates.oneWeekAgo | twDateTime:$ctrl.date.locale }}
