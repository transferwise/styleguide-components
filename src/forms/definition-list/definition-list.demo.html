Definition List

tw-definition-list is a companion to tw-fieldset, it shares the same interface, and same dynamic requirements format, allowing us to display the data model we generated using a fieldset.

You can also specify the layout, which defaults to vertical

Vertical layout 

<tw-definition-list
  legend="Definition List"
  fields="$ctrl.fields"
  model="$ctrl.model"
  locale="en-GB">
</tw-definition-list>

Justified layout 

<tw-definition-list
  legend="Definition List"
  fields="$ctrl.fields"
  model="$ctrl.model"
  layout="'justified'"
  locale="en-GB">
</tw-definition-list>

Horizontal layout 

<tw-definition-list
  legend="Definition List"
  fields="$ctrl.fields"
  model="$ctrl.model"
  layout="'justified'"
  locale="en-GB">
</tw-definition-list>
