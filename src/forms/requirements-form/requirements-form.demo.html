Requirements form

The requirements form uses the dynamic form control to build a fully featured form. It accepts a model object which it permutes based on a set of requirements. The requirements object should match that returned by the TransferWise API

Edit configuration
Configuration
<tw-requirements-form
  requirements="$ctrl.requirements"
  model="{{$ctrl.model | json}}"
  on-refresh-requirements="$ctrl.onRefreshRequirements()"
  upload-options="{
    buttonText: 'Choose file...',
    cancelText: 'Choose a different file'
  }"
  validation-messagess="{
    required: 'Required',
    pattern: 'Incorrect format',
    minlength: 'The value is too short',
    maxlength: 'The value is too long',
    min: 'The value is too low',
    max: 'The value is too high'
  }"
  is-valid='{{$ctrl.isValid}}'></tw-requirements-form>
