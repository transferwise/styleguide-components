Tool Tip

tw-tool-tip is an attribute based directive that enables a floating description to be revealed when an element on hover, focus or click. It is configured through data attributes.

tw-tool-tip requires bootstrap.js

Default tool tip - 
<a tw-tool-tip
  title="By default tool tip appears above">
  Default tool tip
</a>

<tw-help-circle-icon size="24">
  tw-tool-tip
  title="Use data-placement to alter position"
  data-placement="bottom"></tw-help-circle-icon>
