TW Pop Over

tw-pop-over is an attribute based directive that enables a floating description to be revealed when an element is clicked or focused. It is configured through data attributes.

tw-popover requires bootstrap.js

Simple click based - On hover, with custom position - With HTML content 
<a tw-pop-over
  title="Popover title"
  data-content="Full description copy, explaining in more detail">
  Simple click based
</a>

<a tw-pop-over
  title="Popover on hover"
  data-placement="bottom"
  data-trigger="hover"
  data-content="Positioned beneath the target element">
  On hover, with custom position
</a>

<a tw-pop-over
  title="Injected into .popover-title"
  data-content-html="true"
  data-content="HTML enabled <a href='/path'>content</a>, do not use with user entered data">
  With HTML content
</a>
